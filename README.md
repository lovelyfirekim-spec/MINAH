# 인바디 UXUI 개선 프로젝트

이 프로젝트는 인바디(InBody) 앱의 사용자 경험(UX)과 사용자 인터페이스(UI)를 현대적이고 세련된 디자인으로 개선하는 작업입니다.

## 주요 페이지
- `index.html`: 메인 대시보드 및 건강 지표 요약
- `report.html`: 상세 결과 리포트
- `inbody.html`: 인바디 측정 상세 데이터
- `graph.html`: 변화 그래프 확인
- `goal.html`: 목표 설정 및 관리

## 기술 스택
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript

## 폴더 구조
- `css/`: 스타일 시트 파일
- `assets/`: 이미지 및 기타 리소스
